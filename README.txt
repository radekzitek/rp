Virtual Environment
-------------------

python -m venv .venv

this creates the virtual environment .venv

source .venv/bin/activate

this activates the virtual envioronment


Modules
-------

python -m pip install matplotlib

this installs matplotlib (and numpy)

PS: numpy needs some binary stuff to be installed using "sudo apt-get install libatlas-base-dev"
